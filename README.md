# MusicianSuite
A suite of functions for the average musician

Dependencies needed:

- PySide2
- PyQt5 (only for QSharedMemory, waiting for PySide2 bindings)
- soundcard
- numpy
- wavio
- pydub
- sqlite3
- Pillow

To use:
```
python MusicDatabase.py
```
Executable to be compiled soon. Only tested in Windows 10.
